# Telegram alarm bot
An easy way to manage tasks and reminders in telegram.
## Features
- [ ] Multi-language
- [ ] Supports both Date and Duration to add tasks
- [ ] Supports diffrent timezones
- [ ] Supports Gregorian and Hijri Calenders
- [ ] daily reminders
- [ ] Tasklists
